# Okiosk
시각장애인과 노인을 위한 새로운 키오스크 및 보조 어플 제작 

---
## 제작목적
1. 시각장애인을 위한 간편한 키오스크 주문
2. 노인을 위한 키오스크 편의성 증대
3. 어플 연동을 통한 간단한 주문
